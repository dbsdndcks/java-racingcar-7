# java-racingcar-precourse

# 문자열 덧셈 계산기

## 기능 요구 사항

### 1.입력 요구 사항
#### 입력 공통 요구사항
- 잘못된 입력시 `IllgalArgumentException`을 발생시키고 해당 부분부터 다시 입력받는다.

#### 1-1. 자동차 입력
- 자동차 이름을 입력 받는다 (이름은 쉼표(,) 기준으로 구분)
- 자동차 이름은 5자 이하여야한다.

#### 1-2. 경기 횟수 입력
- 경기 횟수를 입력받는다.

### 2.출력 요구 사항

#### 2-1. 자동차 경주 기록 출력
- 경기 횟수 만큼의 경기 기록을 출력해야한다.

#### 2-2. 우승자 출력
- 우승자가 여러 명일 경우 쉼표(,)를 이용하여 구분한다.

### 3.자동차 경주 기능
- (0~9) 사이의 무작위 값을 구한후 값이 4이상일 경우 자동차를 전진한다.
